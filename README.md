# 红腾电气电缆计算器 - PWA移动应用

## 项目简介

这是一个专业的电缆计算器Progressive Web App (PWA)，专为电气工程师和技术人员设计，用于计算中频熔炼设备电源输入电缆的合适规格。应用完美兼容鸿蒙系统及其他主流移动操作系统。

## 功能特点

### 🔧 核心功能
- **智能电缆计算**: 基于功率、电压、长度等多参数计算
- **载流量分析**: 考虑环境温度和安装方式的校正
- **电压降计算**: 精确计算实际电压降百分比
- **电缆选型建议**: 专业推荐适合的电缆型号和规格

### 📱 移动端优化
- **响应式设计**: 完美适配手机、平板等设备
- **触摸友好**: 针对触屏设备优化的界面交互
- **离线使用**: 支持离线计算，无需网络连接
- **快速安装**: PWA技术，一键安装到桌面

### 🚀 鸿蒙系统兼容
- **原生支持**: 鸿蒙系统原生支持PWA应用
- **分布式能力**: 支持鸿蒙多设备协同功能
- **性能优化**: 针对鸿蒙Web引擎优化
- **安全可靠**: 遵循鸿蒙应用安全规范

## 技术架构

### 前端技术栈
- **HTML5**: 语义化标记和现代Web API
- **CSS3**: 响应式设计和动画效果
- **JavaScript ES6+**: 模块化编程和异步处理
- **PWA**: Service Worker和Web App Manifest

### 核心算法
- **电流计算**: I = P / (√3 × U × cosφ)
- **电压降计算**: ΔU = (√3 × I × L × ρ × 100) / (S × U)
- **载流量校正**: 考虑温度系数和安装方式系数

## 安装部署

### 方式一：本地部署（推荐）

1. **下载项目文件**
   ```bash
   # 假设项目已下载到本地目录
   cd cable_calculator_app
   ```

2. **启动本地服务器**
   ```bash
   # 使用Python启动
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve .
   
   # 或使用PHP
   php -S localhost:8000
   ```

3. **访问应用**
   - 打开浏览器访问: `http://localhost:8000`
   - 在手机浏览器中访问相同的地址

4. **安装到桌面**
   - 浏览器会自动提示安装PWA
   - 点击"安装"或"添加到主屏幕"
   - 即可像原生应用一样使用

### 方式二：云端部署

1. **上传到Web服务器**
   - 将所有文件上传到Web服务器根目录
   - 确保服务器支持HTTPS（PWA要求）

2. **配置HTTPS**
   - 申请SSL证书
   - 配置HTTPS访问

3. **测试访问**
   - 通过域名访问应用
   - 测试PWA安装功能

## 故障排除

### 常见问题

**Q: PWA无法安装？**
A: 确保使用HTTPS协议访问，且浏览器支持PWA功能。

**Q: 离线功能不工作？**
A: 检查Service Worker是否正确注册，清除浏览器缓存后重试。

**Q: 计算结果不准确？**
A: 检查输入参数是否正确，确保符合实际工程条件。

**Q: 在鸿蒙手机上无法使用？**
A: 确保使用鸿蒙系统内置浏览器或支持PWA的第三方浏览器。

### 技术支持

如遇到技术问题，请：
1. 检查浏览器控制台错误信息
2. 确认网络连接正常
3. 尝试清除浏览器缓存
4. 重新安装PWA应用
